IO多路转接--poll
发表于2020-10-03|更新于2020-10-03|unix系统高级编程
|阅读量:

​

poll函数类似于select，但是程序接口不同，poll函数可用于任何类型文件的文件描述符。与select不同，poll以文件描述符为单位组织事件。

fdarray 是监听文件描述组的起始地址，nfds为监听个数，timeout设置为等待时间。
struct pollfd{
int fd; //文件描述符
short events; //对文件描述符感兴趣的事件
short revents; //文件描述符监视结果的事件 return events
}
